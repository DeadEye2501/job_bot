# Job Bot

### Быстрый старт

1. Создайте файл `.env`:
Получить API_ID и API_HASH: https://my.telegram.org/apps

2. Добавьте резюме:
`files/cv.pdf`

3. Запустите:
`docker-compose up -d`

4. Соберите информацию:
Добавьте бота в интересующие группы и запустите.
Бот соберёт о их информацию, после чего поставьте `is_active=True` в тех группах, которые хотите обрабатывать.

5. Настройте бота:
Добавьте записи в `Answers` и `Filters`.
